Payslip Excel Report

This Module have the functionality by which user can create payslips for employees.
And also Download the Excel Payslip Report of an Employee.


Open Payroll App -> Click on the check box and select any one Payslip [You can choose multiple users as well]






After selecting any one payslip then click on the "Action" button.




After clicking on the Action button then click on the "Payslip Excel Report" print button.




Now a wizard will appear then click on the "Print Excel Report" in the wizard




After clicking on the Print Excel Report a download button will appear as "Payslip Report.xls" click on that.




Now you can open the downloaded excel file and you can see your Excel Payslip Report as shown in the figure.
